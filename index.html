개인정보처리방침

최종 업데이트: 2025년 3월

똥퍼(이하 "앱")는 사용자의 개인정보를 중요하게 생각합니다. 본 방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

1. 수집하는 정보

  • 위치 정보 — 주변 공중화장실 및 편의점을 검색하기 위해 기기의 GPS 위치(위도·경도)를 수집합니다. 위치 정보는 외부 API 호출 시 검색 기준점으로만 사용되며, 별도 서버에 저장되지 않습니다.
  • 기기 내 저장 데이터 — 검색 결과 캐시 및 앱 설정이 기기 로컬 저장소(AsyncStorage)에만 저장됩니다. 외부로 전송되지 않습니다.

2. 정보의 사용 목적

  • 현재 위치 기반 공중화장실·편의점 검색 및 거리 표시
  • 지도 렌더링 및 경로 안내
  • 반복 검색 속도 향상을 위한 로컬 캐시

3. 제3자 서비스

앱은 아래 외부 서비스에 위치 좌표 등 최소한의 데이터를 전달합니다.

  • 카카오 API — 지도 렌더링, 장소 검색, 주소 변환에 사용됩니다. 카카오 개인정보처리방침
  • 공공데이터포털 (data.go.kr) — 정부 제공 공중화장실 데이터 조회에 사용됩니다. 공공데이터포털 개인정보처리방침
  • 국토교통부 철도 정보 (KRIC) — 지하철 화장실 정보 조회에 사용됩니다.

4. 광고

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 광고 제공을 위해 기기 식별자 등을 수집할 수 있습니다. Google 개인정보처리방침을 참고하세요.

5. 정보 보관 및 파기

앱이 기기에 저장하는 캐시 데이터는 7일 후 자동 만료됩니다. 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

6. 아동 개인정보 보호

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.
